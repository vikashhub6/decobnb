# DecorBnB

A full-stack three-sided marketplace for event decoration services. Customers can browse, search, and book decorated spaces for weddings, birthdays & special occasions. Hosts can list and manage their decoration services. Young workers looking for opportunities can register and connect with decoration shops to find work.

## Tech Stack
- **Frontend:** EJS, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Others:** Multer, Express-Session, connect-mongodb-session

## Features
- User Authentication (Signup/Login)
- Browse & Search Events
- View Event Details
- Add to Favourites
- Host Dashboard
- Image Upload
